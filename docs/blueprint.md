# **App Name**: InvestView

## Core Features:

- Balance Display: Display the current investment balance in a prominent card on the home dashboard.
- Investment Chart: Visualize investment data using a chart, displaying market trends, deposits, and withdrawals.
- Deposit Functionality: Enable users to deposit funds into their investment accounts.
- Withdrawal Functionality: Enable users to withdraw funds from their investment accounts.
- Investment Overview: Provide an overview of different investment types such as crypto and stocks.

## Style Guidelines:

- Primary color: Deep sky blue (#3498DB) to convey trust and stability in financial data.
- Background color: Light gray (#ECF0F1), almost white, to give the charts a clean, uncluttered appearance.
- Accent color: Emerald green (#2ECC71), as an analogous color to sky blue, that is a vibrant hue to indicate positive movements and gains.
- Body and headline font: 'Inter' sans-serif for clear and modern readability.
- Use a consistent set of financial and data-related icons for clarity.
- Clean, card-based layout with clear separation of information sections.
- Subtle animations for transitions and data updates to improve UX.
